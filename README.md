# Little Garden of Automations 🌱

Welcome to my **Little Garden of Automations** – a collection of scripts, helpers, and little tools that make life in my home lab, work, and personal setups easier.

This repository is all about **automation and efficiency** across platforms like:

- **Linux** – system scripts, maintenance, and helpers  
- **Proxmox** – container & VM management, backups, mounts  
- **Windows** – automation, setup scripts, and utilities  

> Think of this as a small garden where every script has its place – practical, reusable, and sometimes a bit quirky 😎

## Goals

- Keep all my automation scripts in **one place**.  
- Make it easy to **reuse and adapt** for different environments.  
- Document usage and purpose clearly so I (and others) can quickly pick up a script.
- Help others in the process

- How to configure it
